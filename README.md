## Features

- Automatically swaps damaged Mending tools into offhand
- Returns fully repaired items to their original slot
- Ignores main-hand weapon so it self-mends during combat
- Designed for XP spawner farms and grinders

## Installation

1. Download the latest release from the **Releases** page
2. Place the `.jar` into your `rusherhacks/plugins` folder
3. Launch Minecraft with RusherHacks installed
4. Enable the AutoMend module
